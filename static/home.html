Management Tools for PostgreSQL

Feature rich | Maximises PostgreSQL | Open Source

The Performance Farm is an Open Source administration and management toolset focused on benchmarking and analysing PostgreSQL performance. It includes a web interface for visualizing results, tools for running benchmark suites, comparing performance across commits or systems, and much more.

The tool is designed to answer the needs of developers tracking performance regressions, DBAs evaluating hardware or configuration changes, and system administrators monitoring performance over time.
